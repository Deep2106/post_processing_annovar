# Batch Post-Annotation Excel Generator

A PED-driven batch pipeline that transforms ANNOVAR hg38_multianno.csv output into structured, clinician-ready Excel workbooks for whole-exome sequencing cohort analysis. Given a PED file, the script auto-detects all family structures — singletons, duos, trios, and quad/multi-child families — and processes each proband independently. For each sample it performs OMIM and HPO gene lookups, applies GEL panel annotation, computes inheritance patterns (De Novo, Maternal, Paternal, Familial) by variant intersection against parental data, and generates gnomAD v4, UCSC, and DECIPHER hyperlinks with correct indel coordinate handling. Output is one .xlsx workbook per proband containing up to six sheets: Raw_Data, DeNovo_Het (trio/duo only), rare_Hom, Comp_Het, Het, and Synonymous. The script supports both legacy (A/B/C) and explicit (P/F/M/D) role-letter conventions in PED SampleIDs, runs interactively with a pre-flight confirmation summary, and captures all output to a timestamped log file. Optional arguments include paths to OMIM, HPO, GEL panel, and an indexed hg38 FASTA for exact indel variant URLs.
